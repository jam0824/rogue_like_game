# HUD
## HUD画面上部
* 画面の左上にHPバーが表示されている。
    * HPバーの中央(HPバーの長さが100pxだったら50pxが中央)には100/100のように現在HP/最大HPの数字が表示される
    * HPバーはHPがmax100なら100px、200なら200pxである。最大HPによりバー全体が伸びる。
    * 文字は黒で縁取りする
    * 参考とするmockはmock/hud_hp_bar_mock.html
* HPの下には「LV 1」といったようにレベルが表示される。文字列は黒で縁取りする
* バフが発生している場合はLVの下にバフのアイコンが横に並ぶ
* デバフが発生している場合は、バフの下に状態異常を示すアイコンが横に並ぶ
* 画面上部右側には「$1,000」のように、文字列で現在持っているお金が表示される。
    * 文字は黒で縁取りする
## HUD画面下部
* 画面下部の中央にはアイテムを簡単に使用するための簡易アイテムインベントリがある
    * 8個まで表示できるインベントリである
    * 登録されたアイテムをクリックすると該当のアイテムを使用できる
    * 参考とするmockはmock/hud_item_box_mock.html
* 簡易アイテムインベントリの横には道具袋アイコンがある
    * 道具袋アイコンをタップするとアイテムインベントリウィンドウが開く。
    * 以下のアイコンを使用する
        * ui/hud/icon_open_inventory_window.png

## HUDトースト（v1実装確定）
* 簡易インベントリ上部にトースト文言を表示する
    * 例：「アイテムを捨てました。」「道具袋がいっぱいで拾えない。」
* トーストは一定時間表示後に自動で消える（約1.8秒）
* 新しいトーストが出た場合は表示時間タイマーをリセットする

# アイテムイベントリウィンドウ
    * ラン内の取得した道具が全て表示されるウィンドウである
    * 参考とするmockはmock/inventory_window_mock.html
    * 初期状態では最大10個まで道具を入れることができる
    * 最大数は地上の施設で開放することができる
    * 道具を選択すると右側にアイテム画像と名前、説明、効果が表示される
        * 表示項目はTODO
    * 多言語対応のため、それぞれはlanguage_keyで管理される
    * アイテム選択後に「USE」ボタンでそのアイテムを使うことができる
    * アイテム選択後に「DROP」ボタンでそのアイテムを地面に捨てることができる
    * DROPは1回につき1個を捨てる
    * DROP先は「プレイヤー近傍のwalkableタイル」に配置する
        * プレイヤー足元タイルには置かない
        * 既に床アイテムがあるタイルには重ねない
    * 複数アイテムを捨てた場合は、アイテムは重なることはない。アイテムがない地面に置かれる。
    * DROPしたアイテムはワールドに表示され、接触で再取得できる
    * 床アイテム表示は「画像表示」とする
    * アイテムインベントリウィンドウは右上の×ボタン、またはウィンドウ外をタップすることで閉じることができる
