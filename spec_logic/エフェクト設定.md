# エフェクト設定
* エフェクトの設定はdb/effect_db/に保存されている
* 攻撃hit時のエフェクトは、攻撃が当たった場所を中心としてエフェクトを再生する。
* 攻撃スタート時のエフェクトは、攻撃を出す場所を中心としてエフェクトを再生する。

# key設定
- id : weaponやskillから呼び出す際に使われるid
- effect_file_name : エフェクト画像が保存されているパス
- width, height : エフェクトの1コマ当たりのサイズ
- animation_fps : エフェクトの再生速度
- animation_direction : エフェクト画像全体でコマがどちらに並んでいるか
    - horizontal : 画像の横方向にアニメーションのコマが並んでいる。アニメーションのコマ数は画像全体のwidth/1コマ当たりのwidthで求められる
    - vertical : 画像の縦方向にアニメーションのコマが並んでいる。アニメーションのコマ数は画像全体のheight/1コマ当たりのheightで求められる
- scale : エフェクトの拡大率
- blend_mode : 画像の合成モード
    - normal : 通常の画像合成
    - add : 加算合成
- loop : アニメーションをループするか。falseの場合はアニメーションの全コマ再生が終わったらエフェクトは消える。trueの場合はアニメーションはまた最初に戻って再生を続ける。
