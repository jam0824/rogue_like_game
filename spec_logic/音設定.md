# 音設定
* db/sound_db/sound_db.jsonにkeyと音ファイルの設定が記載されている
* seは設定されているkeyを使い、sound_db.jsonに設定されているseを鳴らす。

# key設定
* se_key_open_chest : 宝箱を開けたときに慣らす音
* se_key_get_item : アイテムを取得したときに慣らす音
* se_key_put_item : アイテムを捨てたときに慣らす音
* se_key_player_get_damage : プレイヤーがダメージを受けたときに鳴らす音
* se_key_enemy_death : 敵が死んだときに慣らす音
* weapon : weapon_dbに設定されたse_keyから呼び出され、その音を鳴らす。
* skill : アイテムの使用やスキルの使用時にその音を鳴らす。
