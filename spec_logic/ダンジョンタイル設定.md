# ダンジョンの構成DB
- ダンジョンを構成する設定、map_tipの画像ファイルはdb/dungeon_dbの中にjson形式で保存されている
- map_tipのwidth, heightはゲームを通して32pixelである。
- wall_heightは例のB, F, Gにあたる壁の高さである。書かれている数字はタイル数である。よって3タイルの場合は3*32=96pxである
- tip_set_root_pathはmap_tipの画像が保存されているフォルダのpathである。
- tip_set
  - 配置の例にある記号をkeyとしている。
  - tileはwalkableのtileである。
  - 画像は配列で記載されており、配列内に複数の画像が記載されている場合はランダムでそのどれかを配置する
