# 入力・ゲームパッド操作仕様

## 目的
- 本編UI（地上/ダンジョン/インベントリ/保管庫）をゲームパッドで一通り操作できるようにする。
- キーボード/ゲームパッドの割り当てをJSONコンフィグで変更できるようにする。

## 対応範囲
- 対応: 地上施設選択、ダンジョン内移動、宝箱開封、ポーズ、インベントリ（ITEM/WEAPON/CHIP）、武器スキル設定モーダル、保管庫UI。
- 非対応: debug-panel（Seed入力、再生成などデバッグ専用UI）。

## コンフィグファイル
- パス: `db/input_config/input_bindings_v1.json`
- バージョン: `input_bindings_v1`
- 反映タイミング: ページ再読み込み時

### スキーマ（概要）
- `version: string`
- `keyboard: { <action>: string[] }`
- `gamepad: { move_axis: { x: number, y: number, deadzone: number }, <action>: number[] }`
- `action` は以下を使用する:
  - `move_up`, `move_down`, `move_left`, `move_right`
  - `interact`, `pause`, `inventory_toggle`
  - `ui_up`, `ui_down`, `ui_left`, `ui_right`, `ui_confirm`, `ui_cancel`
  - `ui_tab_prev`, `ui_tab_next`
  - `quickslot_1` ～ `quickslot_8`

## デフォルト割り当て

### キーボード
- 移動: `WASD` + 矢印キー
- 調べる/開ける: `E`
- 一時停止: `Esc`
- インベントリ開閉: `I`
- UI決定: `Enter` / `Space`
- UIキャンセル: `Esc`
- タブ切替: `Q`（前） / `E`（次）
- クイックスロット: `1` ～ `8`

### ゲームパッド（standard mapping）
- 左スティック: 移動（`axes[0]`,`axes[1]`、deadzone=0.25）
- D-pad: UI移動（上下左右）
- `A(0)`: 決定 / interact
- `B(1)`: キャンセル
- `Y(3)`: インベントリ開閉
- `LB(4)` / `RB(5)`: タブ切替（前/次）
- `Start(9)`: 一時停止
- クイックスロット: 初期値は未割り当て（空配列）

## 画面別の入力ルール
- ダンジョン通常時:
  - `move_*` / 左スティックで移動
  - `interact` で隣接宝箱（1タイル以内）を開く
  - `quickslot_*` で対応スロットを使用
  - `pause`, `inventory_toggle` は常時有効（遷移中/死亡シーケンス中を除く）
- UI表示中（地上/インベントリ/保管庫）:
  - `ui_*` でフォーカス移動
  - `ui_confirm` でフォーカス要素を実行（`click`）
  - `ui_cancel` は文脈に応じて閉じる（スキルモーダル→インベントリ→保管庫）
  - `ui_tab_prev/next` でタブ循環（インベントリ/保管庫）

## 優先順位
- 方向入力は「方向入力 > ポインター追従」。
- UI表示中はゲームプレイ移動へ入力を流さず、UIナビゲーションを優先する。
- シーン/フロア遷移中はUIナビゲーションとゲームプレイ入力を無効化する。

## 確認手順
- `npm run unit`
- `npm run test:checks`
- 手動確認:
  - 地上施設選択→ダンジョン遷移
  - ダンジョン移動、宝箱開封、ポーズ
  - インベントリ（ITEM/WEAPON/CHIP）操作
  - 武器スキル設定モーダル操作
  - 保管庫（入出庫/売却/拡張）
