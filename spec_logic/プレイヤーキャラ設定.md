# プレイヤーキャラ設定
* プレイヤーキャラのチップは32x64である
* 壁は通り抜けることができない。
* 壁との当たり判定は上半分の32x32にはない。下半分の32x32に壁との当たり判定がある。
* 敵の攻撃を受けた際の当たり判定はキャラクターと同サイズの32x64である。
* 敵の攻撃を受けた際の当たり判定はスキルによって小さくなる場合がある。

# プレイヤーキャラのチップ設定について
* player_tipはchar_p_hero_m03a.pngである。プレイヤーのアニメーションが全てこのpngにまとまっている
* 32x64の画像が、4行3列でまとめられた画像である。
* 1行目が下移動時、2行目が左移動時、3行目が右移動時、4行目が上移動時のアニメーションである。
* アニメーションは1列目をA、2列目をB、3列目をCとする。その場合アニメーションの順番はA→B→C→B→A→B→C→……とループする

# プレイヤーの移動について
* プレイヤーはマウスクリックがあった場所に一直線に移動する
* 移動途中に壁があった場合、そこで止まる
* マウスを押している間、マウスを追従する
* プレイヤーの移動量が横の方が大きい場合は横のアニメーション、上や下の移動量が多い場合は上移動や下移動のアニメーションをする
* マウス話したらその時点で移動をやめる
